# ISO Date-Time

<p class="or-lede">Full ISO timestamp.</p>

Use for retrieved-at, updated-at, or event timestamps when time is known.

## Wire shape

```json
{
  "$type": "date-time-iso8601",
  "value": "2024-06-01T12:34:56Z"
}
```

## Examples

- `"2024-06-01T12:34:56Z"`
