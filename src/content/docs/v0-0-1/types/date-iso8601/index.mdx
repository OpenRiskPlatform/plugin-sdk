# ISO Date

<p class="or-lede">Full ISO date.</p>

Use when the source provides a complete calendar date in YYYY-MM-DD form.

## Wire shape

```json
{
  "$type": "date-iso8601",
  "value": "2024-06-01"
}
```

## Examples

- `"2024-06-01"`
