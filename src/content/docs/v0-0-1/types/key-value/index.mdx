# Key Value

<p class="or-lede">Labeled extra value.</p>

Used inside $extra to preserve source-specific fields without promoting them into canonical props.

## Wire shape

```json
{
  "$type": "key-value",
  "value": {
    "key": "Dataset",
    "value": "pep"
  }
}
```

## Examples

- `{"key":"Dataset","value":"pep"}`
