# Base64 Image

<p class="or-lede">Base64 encoded image payload.</p>

Use sparingly for embedded images when a URL is not available.

## Wire shape

```json
{
  "$type": "image-base64",
  "value": "data:image/png;base64,..."
}
```

## Examples

- `"data:image/png;base64,..."`
