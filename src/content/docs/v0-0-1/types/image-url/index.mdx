# Image URL

<p class="or-lede">URL to an image.</p>

Use for externally hosted profile images or logos.

## Wire shape

```json
{
  "$type": "image-url",
  "value": "https://example.com/avatar.jpg"
}
```

## Examples

- `"https://example.com/avatar.jpg"`
