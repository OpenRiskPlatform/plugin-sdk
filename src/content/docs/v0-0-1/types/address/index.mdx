# Address

<p class="or-lede">Human-readable address.</p>

Use for postal addresses and source-provided location strings that should render as addresses.

## Wire shape

```json
{
  "$type": "address",
  "value": "Main Street 1, Bratislava"
}
```

## Examples

- `"Main Street 1, Bratislava"`
