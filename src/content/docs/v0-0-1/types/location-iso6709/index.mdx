# ISO 6709 Location

<p class="or-lede">Geographic coordinate string.</p>

Reserved for coordinate values following ISO 6709-style location encoding.

## Wire shape

```json
{
  "$type": "location-iso6709",
  "value": "+48.1486+017.1077/"
}
```

- `"+48.1486+017.1077/"`
