# Number

<p class="or-lede">JSON number.</p>

Use for numeric scores, counters, monetary amounts already normalized as numbers, or credit values.

## Wire shape

```json
{
  "$type": "number",
  "value": 42
}
```

## Examples

- `42`
- `0.87`
