# Partial ISO Date

<p class="or-lede">Partial ISO date.</p>

Use when the source only provides a year or year-month value.

## Wire shape

```json
{
  "$type": "date-partial-iso8601",
  "value": "2024"
}
```

## Examples

- `"2024"`
- `"2024-06"`
